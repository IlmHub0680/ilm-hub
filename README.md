# Ilm Hub - Islamic Educational Platform

Production-ready Phase 1 of Ilm Hub, an Islamic educational platform.

## Quick Start

```bash
npm install
npm run dev
```

Visit: `http://localhost:3000`

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- PostgreSQL (future phases)
- Prisma (future phases)

## Features (Phase 1)

- Responsive homepage with Islamic green/white theme
- Featured books showcase
- How-it-works guide
- Mobile-first design
- Bilingual support (EN/AR) - coming Phase 2

## Project Structure

```
ilm-hub-production/
├── app/
│   ├── page.jsx           # Homepage
│   ├── layout.jsx         # Root layout
│   └── globals.css        # Global styles
├── package.json

```
